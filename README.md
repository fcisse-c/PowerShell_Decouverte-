# PowerShell_Decouverte

## Liste des commandes Unix et leurs équivalents PowerShell

| Commande Unix | Commande PowerShell |
|----------------|---------------------|
| `cp`           | `Copy-Item`         |
| `rm`           | `Remove-Item`       |
| `cd`           | `Set-Location`      |
| `mkdir`        | `New-Item -ItemType Directory` |
| `man`          | `Get-Help`          |
| `history`      | `Get-History`       |
| `alias`        | `Set-Alias`         |
| `cat`          | `Get-Content`       |
